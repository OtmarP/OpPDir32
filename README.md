# OpPDir32
Command-Line-Tool Path-Dir

Lists the Path-Environment-Variable

Search for a File (Program) in the Path-List

# Sample
    C:\>OpPDir32
    OpPDir v:1.0.16.1221.R
    ----------------------------------------------------------------------
    Current = C:\
    PATH    = C:\WINDOWS\system32;C:\WINDOWS;C:\S
    Cnt Len = Path
    ----------------------------------------------------------------------
      0   3 = C:\ (=CurDir)
      1  19 = C:\WINDOWS\system32
      2  10 = C:\WINDOWS
      3   4 = C:\S
    ----------------------------------------------------------------------

Search for cmd.exe

    C:\>OpPDir32 cmd.exe
